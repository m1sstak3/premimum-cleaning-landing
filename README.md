## 🚀 Ключевые особенности

Лендинг разработан с упором на эстетику, скорость загрузки и удобство пользователя, чтобы максимизировать конверсию в заявку.

* 💎 **Современный UI/UX:** Чистая, "воздушная" эстетика, эффекты глассморфизма (Glassmorphism) и продвинутая стилизация через CSS-переменные.
* 📱 **Полная адаптивность:** Идеальное и бесшовное масштабирование под экраны мобильных устройств, планшетов и широкоформатных мониторов (Mobile First).
* ⚡ **Экстремальная производительность:** Использование исключительно нативного JavaScript (без тяжелых библиотек вроде jQuery) и минимальное количество сторонних ресурсов.
* 🎯 **Фокус на конверсии:** Закрепленное навигационное меню (Sticky Header), плавная прокрутка к блокам (Smooth Scroll) и интуитивно понятная интерактивная форма для сбора заявок.

## 🛠️ Технологический стек и Архитектура

Проект демонстрирует уверенное владение фундаментальными веб-технологиями и современными подходами к верстке:

* **HTML5:** Строгая семантическая структура для доступности (a11y) и SEO-оптимизации.
* **CSS3:** Использование БЭМ-подобной методологии нейминга классов, Flexbox, CSS Grid и Custom Properties (CSS-переменных) для легкой кастомизации темы.
* **Vanilla JavaScript (ES6+):** Чистая логика взаимодействия с DOM-деревом и обработка событий без оверхеда сторонних фреймворков.
* **Иконки:** Легковесная библиотека `Lucide Icons`.

## 💻 Установка и использование

Это классический статический веб-сайт (Zero Dependency). Сложные инструменты сборки (Node.js, Webpack, Vite) не требуются.

1. Склонируйте репозиторий на свой компьютер:
```bash
git clone [https://github.com/m1sstak3/premium-cleaning-landing.git](https://github.com/m1sstak3/premium-cleaning-landing.git)
```
2. Откройте файл `index.html` в любом современном веб-браузере.

*(💡 Для комфортной локальной разработки рекомендуется использовать расширение **Live Server** в VS Code).*

> ⚠️ **Примечание для проверяющих:** Отправка формы симулируется с помощью JavaScript исключительно для демонстрации в портфолио. В текущей версии реальные письма на сервер не отправляются.
